# 🌟 Student Survival

A cute little student-help website made with love 💖

## ✨ About This Project
Student Survival is a simple and friendly webpage for students.
It is lightweight, clean, and easy to open anywhere 📚

## 🎀 Features
- Simple single-page layout 🧁
- Fast and lightweight ⚡
- Beginner-friendly code 👩‍💻
- Easy to host on GitHub Pages or Vercel 🌍

## 🛠 Tech Stack
- HTML5 🧱
- CSS3 🎨
- JavaScript (if added) 🧠

## 🚀 Run Locally
1. Download or clone this repository
2. Open the project folder
3. Open index.html in your browser

Or using VS Code Live Server/Five Server:

    Open index.html
    Click Go Live

## 🌐 Deployment
You can deploy this project using:
- GitHub Pages 💙
- Vercel ▲
